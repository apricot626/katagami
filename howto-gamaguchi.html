がまぐちポーチの作り方

カチッと閉まる口金が魅力のがまぐちポーチ。お使いの口金サイズを入力すると、それに合った布の型紙を自動計算します。小銭入れやアクセサリー入れにおすすめです。

  • 紙ひも（口金付属または別売り）
  • 目打ち・木工用ボンドまたは専用接着剤

2. 型紙の作り方（カタガミで自動生成）

カタガミの型紙ツールを使うと、仕上がりサイズを入力するだけで縫い代付きの型紙を自動計算してA4に分割印刷できます。

  1. ツールを開く
    型紙ツールを開き、上部タブの「バッグ」→「がまぐちポーチ」を選択します。
  2. サイズを入力する
    お使いの「口金の幅（外幅）」「口金の高さ」「ポーチの深さ」を入力します。布幅は口金幅×1.6で自動計算されます。「角丸タイプ」をオンにすると口金の丸みに合います。
  3. 縫い代を設定する
    下部スライダーで縫い代幅を選びます（初心者は 1.0〜1.5 cm がおすすめ）。
  4. 印刷する
    「印刷する（A4・実寸）」ボタンを押し、印刷ダイアログで倍率 100%・用紙に合わせるをオフにして印刷します。 1枚目の案内シートにある50mm校正枠を定規で確認してください。

3. 布の裁断

  1. 型紙を布に写す
    型紙を布の裏に写します。表布には接着芯を貼っておくと、口金にしっかり収まります。
  2. 裁断する
    表布2枚・内布2枚を裁ちます。型紙の上端にある破線が口金の取り付け位置です。
  3. 口金で形を確認
    裁断前に、実際の口金を型紙の上端にあてて、布の幅・カーブが合っているか確認すると失敗が減ります。

4. 縫い方

4-1. 袋を縫う

  1. 表布2枚を中表に合わせ、口金取り付け位置（破線）より下の脇と底を縫います。破線より上（口金が付く部分）は縫いません。
  2. 内布も同様に、片側に返し口を残して縫います。
  3. マチを付ける場合は、表布・内布とも底角を三角につまんで縫います。

4-2. 表袋と内袋を合わせる

  1. 表袋を表に返し、内袋（裏のまま）の中に入れて中表に重ねます。
  2. 口金が付く開口部のカーブをぐるりと縫い合わせます。
  3. 内袋の返し口から全体を表に返し、返し口を閉じます。口の形を整えます。

4-3. 口金を付ける

  1. 口金の溝に木工用ボンド（または専用接着剤）を入れます。
  2. 袋の口を目打ちで口金の溝に押し込みます。中心→両端の順に入れるとずれません。
  3. 溝と布の間に紙ひもを押し込んで固定します。
  4. 当て布をして、ペンチで口金の端を軽く締めます（金具を傷めないよう布を当てる）。乾かしたら完成です。

✏️ 口金付けは「中心を合わせてから両サイドへ」が鉄則です。ボンドが乾く前に左右のバランスを確認しましょう。

5. コツと応用

  • パンッと張らせたい：表布・内布の両方に接着芯を貼ると、ふっくら張りのあるがまぐちになります。
  • 口金が入らない：布の口が口金の溝に対して厚すぎる場合があります。接着芯を薄手にするか、紙ひもを細いものに変えます。
  • 初めての口金：10.5cm前後の差し込みタイプが扱いやすく、練習に向いています。

型紙はカタガミで無料生成できます。サイズを変えてすぐに試せます。
