# バックログ（機能アイデア・タスク）

思いついた機能や改善をここに溜めておき、区切りで要件定義・憲章に反映します。
GitHub を使い始めたら、各項目を Issue に起票してもよい。

優先度：🔴高 / 🟡中 / 🟢低（暫定）

## 機能アイデア
- 🟡 印刷時の透かしロゴ（出所表示・盗作抑止）（F-27）— 薄い背景透かし、印刷時のみ。ライセンスの「商用は出所表記」と整合
- 🟡 サイズプリセットの子供サイズ追加（F-26の残り）— 身長別（80/90/100…）。既存の大人/犬/スカートは実装済み
- 🟡 採寸プロファイルの保存／呼び出し（端末ローカル）（F-20）
- 🟡 PDF直接ダウンロード（印刷ダイアログを介さない）（F-21）
- 🟡 パターン追加・バリエーション（袖型違い、サイズ展開など）（F-22）
- 🟢 多言語対応

## パターン追加の候補（難易度／区分タグ付き）

難易度：易＝矩形ベースで簡単、中＝曲線少なめ、難＝立体フィット・要仮縫い検証
区分：新規＝新しい型、バリエーション＝既存型のパラメータ違い

| 候補 | 難易度 | 区分 | メモ |
|---|---|---|---|
| シュシュ | 易 | 新規 | 細長い矩形＋ゴム通し |
| ブックカバー | 易 | 新規 | 矩形＋折り返し。文庫/A5でサイズ違い |
| ポーチ | 易 | 新規 | 矩形＋ファスナー位置。マチはトート式を流用 |
| 犬服 マナーベルト | 易 | 新規 | 筒状の帯＋面ファスナー位置。ほぼ矩形 |
| ギャザースカート | 易 | 新規 | 矩形＋ギャザー＋ウエストベルト |
| 長袖/半袖Tシャツ | 易 | バリエーション | 既存Tシャツの袖丈パラメータ違い |
| エプロン | 中 | 新規 | 本体台形＋ひも＋ポケット。曲線は少なめ |
| 犬服 袖付き | 難 | バリエーション | 既存犬服に前足の筒。立体で体型差大 |
| 子供服 | 易〜難 | 要相談 | 子供用Tシャツ等なら既存型＋子供プリセットで対応（易）。ワンピース/パンツ等は新規で難 |

> メモ：「子供服」はサイズプリセット（F-26）の子供サイズで既存型を流用できる部分が大きい。新規の難しい型は後回しでよい。
> 進め方の目安：矩形ベースの簡単な型で数を増やしてから、子供服・犬服袖付きなど重いものを1つずつ。小さな違いはトグル、構造が別物なら新しい型。
