# バックログ（機能アイデア・タスク）

思いついた機能や改善をここに溜めておき、区切りで要件定義・憲章に反映します。
GitHub を使い始めたら、各項目を Issue に起票してもよい。

優先度：🔴高 / 🟡中 / 🟢低（暫定）

## 機能アイデア
- 🟡 採寸プロファイルの保存／呼び出し（端末ローカル）（F-20）
- 🟡 PDF直接ダウンロード（印刷ダイアログを介さない）（F-21）
- 🟡 パターン追加・バリエーション（袖型違い、サイズ展開など）（F-22）
- 🟢 多言語対応

## パターン追加の候補
- ギャザースカート / エプロン / 子供服
- 犬服：袖付き（前足を通す筒）、マナーベルト対応
- 小物：ポーチ、ブックカバー、シュシュ

## コンテンツ（収益化の前提）
- 🔴 作り方の読み物記事（初回テーマ未定：巾着 or トート）（F-23）
- 🔴 プライバシーポリシー・問い合わせページ（F-24）

## 計測・運営
- 🔴 アクセス解析の導入（ツール未定）（F-25 / 5.5）
- 🟡 「印刷実行」「パターン選択」のイベント計測

## 改善メモ
- 🟢 小さな矩形ピース（蝶ネクタイの本体・バンド等）は、タイル印刷せず寸法だけ読んで直接裁断する案内があると親切
- 🟢 スマホ印刷時の「A4・フィット無効」案内表示

## 完了済み
- ✅ 蝶ネクタイの型紙を追加（こども・小物）
- ✅ トグル不具合の修正（首バンド有無・パネルの「わ」）
- ✅ 印刷の対応端末を両対応に確定
- ✅ ライセンス確定（コードMIT／型紙は個人利用自由・商用は出所表記）
