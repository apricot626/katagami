# カタガミ（KATAGAMI）

数値を入力するだけで、洋裁の型紙を自動製図し、**A4分割で実寸印刷**できるブラウザツール。
大人・子供の服、巾着などの小物、トートバッグ、犬服などを、モード別に切り替えて生成できます。

ビルド不要・サーバー処理なし。`index.html` を開くだけで動く静的アプリです。

## 特徴

- 数値（cm）入力でリアルタイムにSVG製図（裁ち切り線・出来上がり線・布目線・「わ」・合印つき）
- 縫い代の調整、出来上がり線の表示切替
- A4タイル分割印刷。重ね代と貼り合わせグリッドで位置合わせがしやすい
- 1枚目の案内シートに **50mmの校正枠** を出力。印刷倍率を必ず検算できる
- 採寸値はブラウザ内だけで処理（サーバー送信なし）

## モードとパターン

| モード | 主なパターン |
|---|---|
| 大人服 | Tシャツ、Aラインスカート |
| 子供服 | 子供Tシャツ、ギャザースカート、スタイ、ゴムウエストパンツ、移動ポケット |
| 小物 | 巾着・小袋、コップ袋、ランチョンマット、ティッシュケース、ヘアバンド、蝶ネクタイ |
| バッグ | トートバッグ、ポーチ、サコッシュ、がまぐちポーチ、汎用パネル |
| ペット | 犬服（タンク／袖付き）、マナーベルト |

> Tシャツの袖・犬服・パンツなど体型差の大きいものは「まず仮縫いで合わせる下書き原型」として設計しています。スカート・巾着・バッグ・小物類は寸法どおりに使えます。

## 使い方

1. `index.html` をブラウザで開く
2. モードとパターンを選び、数値を入力する
3. 「印刷する」を押す
4. 印刷ダイアログで **倍率＝100%（実際のサイズ）**、「用紙に合わせる」は**オフ**にする
5. 1枚目の案内シートの50mm枠を定規で測り、実測50mmなら成功。型紙を切り出す

## ローカルでの確認

ビルド不要。ファイルを開くだけで動きます。

```bash
# そのまま開く
open index.html        # macOS（Windowsは start index.html）

# もしくは簡易サーバーで（印刷挙動の確認に便利）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```
