# カタガミ（KATAGAMI）

数値を入力するだけで、洋裁の型紙を自動製図し、**A4分割で実寸印刷**できるブラウザツール。
人のTシャツ・子供用の小袋（巾着）・手提げ・犬服などを、モード別に切り替えて生成できます。

> 個人開発・学習プロジェクト。「自分で使う道具を作る」ことを起点に、Git管理 → 公開 → 計測 → 運営 → 収益化までを一通り体験することを目標にしています。

## 特徴

- 数値（cm）入力でリアルタイムにSVG製図（裁ち切り線・出来上がり線・布目線・「わ」・合印つき）
- 縫い代の調整、出来上がり線の表示切替
- A4タイル分割印刷。重ね代と貼り合わせグリッドで位置合わせがしやすい
- 1枚目の案内シートに **50mmの校正枠** を出力。印刷倍率を必ず検算できる
- サーバー処理なし。採寸値はブラウザ内だけで処理

## モードとパターン

| モード | パターン |
|---|---|
| 服（人） | Tシャツ、Aラインスカート |
| こども・小物 | 巾着・小袋、蝶ネクタイ |
| バッグ | 手提げ（トートバッグ）、汎用パネル |
| ペット | 犬服（タンク） |

> Tシャツの袖と犬服は体型差が大きいため「まず仮縫いで合わせる下書き原型」として設計しています。スカート・巾着・手提げ・汎用パネルは寸法どおりに使えます。

## 使い方

1. `index.html` をブラウザで開く
2. モードとパターンを選び、数値を入力する
3. 「印刷する」を押す
4. 印刷ダイアログで **倍率＝100%（実際のサイズ）**、「用紙に合わせる」は**オフ**にする
5. 1枚目の案内シートの50mm枠を定規で測り、実測50mmなら成功。型紙を切り出す

## ローカルでの確認

ビルド不要。ファイルを開くだけで動きます。

```bash
# そのまま開く
open index.html        # macOS（Windowsは start index.html）

# もしくは簡易サーバーで（印刷挙動の確認に便利）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## バージョン管理（GitHub・非公開）

このプロジェクトは **非公開（private）リポジトリ** でコードとドキュメントを版管理し、進行管理にも使います（手順は `docs/setup_github.md`）。サイトの**公開（ホスティング）は別フェーズ・別手段**で行います（GitHubの無料プランでは非公開リポジトリからのPages公開ができないため）。

公開する段になったら、非公開リポジトリから無料デプロイできる Cloudflare Pages / Netlify 等、または公開用の別リポジトリを使う想定です。
